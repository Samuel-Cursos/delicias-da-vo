ADICIONE ESTE BLOCO DENTRO DE:
match /databases/{database}/documents {

Não coloque depois das chaves finais.

    // Encomendas de salgados para festas
    match /encomendasFesta/{pedidoId} {
      // O site pode criar uma encomenda.
      allow create: if true;

      // Somente administradores podem ler, editar status ou excluir.
      allow read, update, delete: if isAdmin();
    }

O bloco deve ficar ANTES do bloqueio geral:

    match /{document=**} {
      allow read, write: if false;
    }

Depois clique em Publicar no Firebase Console.


IMPORTANTE — CONTADOR SEQUENCIAL DOS PEDIDOS

Adicione também este bloco dentro de:
match /databases/{database}/documents {

    // Contador usado para gerar DV-0001, DV-0002, DV-0003...
    match /contadores/encomendasFesta {
      allow read, create, update: if true;
      allow delete: if isAdmin();
    }

Ele também deve ficar ANTES do bloqueio geral.
